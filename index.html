Clone

Step1: git clone + url of repository
Step2: ls
step3: cd + Repository name
step4: code . (to get into Visual studio code)

Add

step5: git add + file name (to upload file in repository)
step6: git status (displays the state of the working directory and the staging area)

Commit

step7: git commit -m "(Repository name)"

Status

step8: git status

Push

step9: git push
